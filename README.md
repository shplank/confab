# confab

A chat app for mobile devices built using React Native and Gifted Chat. Expo and Android Studio were used in the development process.

## Required libraries:

- React Native
- React Navigator
- Gifted Chat
- Firebase
- Asynch Storage
- NetInfo

## For development:

- Expo
- Android Studio

## To set up:

```console
git clone https://github.com/shplank/confab.git
cd confab/
npm install
```
